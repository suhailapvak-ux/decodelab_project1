# DecodeLabs Internship Project 1 - To-Do List

Project name: DecodeLabs Internship Project 1 - To-Do List

What it does
- Provides a simple command-line to-do list application that lets you add, view, delete tasks, and save/load them to/from a JSON file (`tasks.json`).

How to run
- Make sure you have Python 3 installed.
- Run the program with:

```
python todo.py
```

Features
- Add a task
- View all tasks (uses `enumerate()` to show numbered list)
- Delete a task by number
- Save tasks to `tasks.json`
- Load tasks from `tasks.json` on startup and via the menu
